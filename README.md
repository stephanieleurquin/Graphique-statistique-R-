# Exercice : Visualisation des notes du second groupe

## Description
Ce projet est un exercice en R pour pratiquer :
- La création de vecteurs
- Les calculs statistiques de base (`mean`, `sum`, `max`, `min`)
- La visualisation des données avec un graphique en barres

Le script principal `notes2_graph.R` crée un vecteur `notes2`, calcule les statistiques, puis affiche un **barplot** avec la moyenne en rouge.

---

## Contenu du projet
- `notes2_graph.R` : script R contenant le code pour créer le graphique et les statistiques
- `notes2_graph.png` : image du graphique généré
- `README.md` : ce fichier d'explication

---

## Comment exécuter le script
1. Cloner ou télécharger ce repository
2. Ouvrir `notes2_graph.R` dans RStudio
3. Exécuter le code (ligne par ligne ou tout le script)
4. Observer les résultats dans la console et le graphique dans l’onglet "Plots"

---

## Exemple de sortie
- Moyenne : 12.83  
- Somme : 77  
- Note maximale : 17  
- Note minimale : 8  

Le graphique affiche chaque note avec une barre verte et la moyenne en ligne rouge.
